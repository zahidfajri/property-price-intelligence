# Property Price Intelligence Dashboard

## Features

- Analyze rental prices by area
- Price summary by room type
- Monthly and yearly rental estimates
- Excel report export
- Supports area names and Speedhome URLs

## Technology Stack

- Python
- Streamlit
- Pandas
- Playwright
- BeautifulSoup
- OpenPyXL

## Deployment Note

Due to Speedhome's cloud IP restrictions, data is scraped locally and stored as CSV files before deployment, following the instructions provided in the assessment email.
